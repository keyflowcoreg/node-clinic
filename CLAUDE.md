# Node Clinic

## Stack
React 19 + Vite 6 + TypeScript + Tailwind CSS v4 + Motion v12

## Commands
- Dev: `npm run dev` (port 3000)
- Build: `npm run build`
- Preview: `npm run preview`

## Design System
- Colors: ivory (#FDFBF7), graphite (#1A1A1A), silver (#C0C0C0)
- Fonts: Space Grotesk (display), Inter (body)
- Pattern: sharp-edge (no border-radius), glass-panel, subtle-shadow
- Micro-copy: Clinical tone, not marketplace

## Structure
- `src/pages/` - Route pages
- `src/components/` - Shared components
- `src/context/` - React contexts
- `src/utils/` - Utility functions
- `src/lib/` - Library helpers
